# Student Performance Dashboard (Tableau)

This project analyzes student exam performance using the "Students Performance in Exams" dataset from Kaggle.

## Tools Used
- Tableau Public
- CSV Dataset

## Key Features
- Calculated fields for Average Score and Pass/Fail
- Subject-wise, gender-wise, and test preparation analysis
- Interactive filters for better exploration
- Insight-based dashboard

## Dataset Source
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams
